Welcome to Pipe, a wrapping tool for emulators made with the explicit purpose of being simpler and more user friendly for those who are new to emulation. This tool helps you turn game files and emulators into simple .exe files that can run on steam. This document outlines how to effectively use Pipe.

Pipe operates in a manner that requires you to already have the game file and the emulator downloaded on your computer. We do not condone piracy, so I recommend that you dump any game files legally obtained into a folder and use those.

In order to use Pipe you must first select the .exe file for the emulator in which you are using. Any settings for the emulator should be set within the emulator itself.

Next, You must select the game file. You are able to select any file type, however it is advised to only select files you know will work on the emulator chosen.

Then, you will set a file path for Pipe to dump your file. Place it somewhere you will easily be able to access during and after compilation to ensure ease of use and retrieval.

Finally, give it a name! Whatever you name your file will be the name of the folder as well as the .exe file.

And... Tada! A new folder should have been created. It contains the following information:
- A .exe file to launch the game
- A .config file (the launcher reads this file, keep it safe!)
- A text file filled with a greeting, some useful resources, next steps, and a dev signature

Once this is created, I would highly recommend reading the next readme file within your new folder as it contains information on next steps to run the .exe on steam. For the uninitiated it will be useful.
